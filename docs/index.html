Presentación del Proyecto

Este sitio web ha sido desarrollado como parte del trabajo práctico de la asignatura Fundamentos de la Ingeniería Informática, perteneciente al Grado en Ingeniería Informática de la Universidad Francisco de Vitoria.

El proyecto tiene como finalidad aplicar de forma práctica los conocimientos básicos adquiridos en diseño web, estructuración de información y uso de tecnologías estándar como HTML y CSS, sin emplear lenguajes de programación ni frameworks avanzados.

Objetivos del Sitio Web

El objetivo principal de esta web es servir como una plataforma personal donde se presenta información académica, profesional y de interés relacionada con el ámbito de la ingeniería informática.

De manera más específica, este sitio busca:

  • Demostrar una correcta estructura de un sitio web estático.
  • Aplicar estilos CSS de forma coherente y uniforme.
  • Facilitar una navegación clara e intuitiva entre las distintas secciones.
  • Presentar contenidos organizados y bien explicados.

Estructura de la Web

La web está dividida en varias secciones claramente diferenciadas, accesibles desde el menú de navegación superior, presente en todas las páginas del sitio.

Cada sección cumple una función concreta:

  • Inicio: visión general del proyecto y sus objetivos.
  • Acerca de mí: información personal, perfil académico y redes profesionales.
  • Mi Grado: descripción detallada del Grado en Ingeniería Informática y sus asignaturas.
  • Sección Temática: desarrollo de un tema de interés relacionado con la informática.
  • Red: enlaces a las páginas web de compañeros.
  • Contacto: formulario sencillo para ponerse en contacto.

Metodología de Desarrollo

El desarrollo del proyecto se ha realizado de forma progresiva, comenzando por la creación de la estructura HTML y posteriormente aplicando estilos CSS para mejorar la presentación visual.

Todo el código se ha gestionado mediante Git y GitHub, lo que ha permitido llevar un control de versiones adecuado y reflejar la evolución del proyecto a lo largo del tiempo.

Tecnologías Utilizadas

Para la realización de este sitio web se han utilizado exclusivamente tecnologías básicas y estándares del desarrollo web:

  • HTML5 para la estructura y el contenido.
  • CSS3 para el diseño visual y la maquetación.
  • Git y GitHub para el control de versiones y publicación del proyecto.
