Presentación

Esta página web forma parte de un proyecto académico de la asignatura Fundamentos de la Ingeniería Informática.

El sitio ha sido diseñado para mostrar conocimientos básicos de desarrollo web, organización de información y diseño visual utilizando HTML y CSS.

Contenido del sitio

A lo largo de las diferentes secciones se puede encontrar información personal, datos sobre el grado universitario, una sección temática específica, enlaces a compañeros y un formulario de contacto.

Objetivo principal

El objetivo principal es ofrecer una web clara, estructurada y visualmente atractiva que sirva como base para futuros proyectos más avanzados.
